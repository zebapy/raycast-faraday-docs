# Faraday Docs

Browse the Faraday API reference documentation from Raycast.

## Overview

This repository contains a small Raycast extension that lets you search and browse the Faraday API reference documentation directly from Raycast.

## Prerequisites

- Node.js (recommend v18+)
- npm or yarn
- Raycast CLI (`ray`) for development and publishing (see Raycast docs)

## Install

Clone the repo and install dependencies:

```bash
git clone https://github.com/zebapy/raycast-faraday-docs.git
cd raycast-faraday-docs/faraday-docs
npm install
```

## Development

- Start development mode (opens the extension in Raycast):

```bash
npm run dev
```

- Build the extension:

```bash
npm run build
```

- Lint and auto-fix issues:

```bash
npm run fix-lint
```

## Usage

Once installed and running in Raycast you can use the `Search Faraday API reference` command to lookup endpoints and documentation within the extension.
